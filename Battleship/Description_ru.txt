Русский:

    Описание проекта

        Для запуска кода введите в консоль "php battleship.php"

        Этот PHP-проект предоставляет функцию, которая создает и отображает сетку, представляющую игровое поле с кораблями,
        размещенными на заданных координатах.
        Размеры сетки определяются параметрами $x (количество столбцов) и $y (количество строк).
        Корабли обозначаются символом "X" на сетке в соответствии с координатами, указанными в массиве $ships.

    Функции

        Функция colle($x, $y, array $ships = []) выполняет следующие действия:

            Проверка входных данных: Проверяет, что $x и $y больше нуля. Если нет, функция завершается без дальнейшего выполнения.

            Инициализация сетки: Создает двумерный массив $setka, заполненный пробелами (" "), представляющий пустое игровое поле.

            Размещение кораблей: Перебирает массив $ships и размещает "X" на сетке по каждой допустимой координате. Координаты за пределами сетки игнорируются.

            Отображение сетки: Выводит сетку с границами, используя ASCII-символы, показывая расположение кораблей.

    Пример использования

        colle(5, 3, [[0, 1], [2, 1], [1, 2], [1, 7]]);

        Этот пример создает сетку с 5 столбцами и 3 строками, размещая корабли на координатах (2,1), (2,1), (1,2) и (1,7).
        Обратите внимание, что координата (1,7) находится за пределами сетки и будет проигнорирована.

        +---+---+---+---+---+

        +---+---+---+---+---+
        |   | X |   |   |   |
        +---+---+---+---+---+
