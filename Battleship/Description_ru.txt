Русский:

    Описание проекта

        Этот PHP-скрипт представляет простую реализацию классической игры "Морской бой" для двух игроков.
        Каждый игрок размещает свои корабли на сетке 5x5 и по очереди пытается потопить корабли противника,
        угадывая их координаты. Игра продолжается до тех пор, пока у одного из игроков не закончатся корабли.
    
    Запуск

        php battleship.php
        Эта команда запустит игру

    Функциональность

        Инициализация сетки

            Игра инициализирует две сетки 5x5 с помощью функции initGrid($x, $y), создавая пустые сетки для каждого игрока.
    
        Размещение кораблей

            Игрокам предлагается разместить два корабля на своей сетке, вводя координаты X и Y.
    
            Функция addShip(&$grid, $x, $y, $posX, $posY) добавляет корабли на сетку, если координаты корректны и позиция свободна.
    
        Игровой цикл

            Игроки по очереди атакуют сетку противника.
        
            Во время хода игрок вводит координаты X и Y для своей атаки.
        
            Функция attack(&$grid, $posX, $posY) проверяет, есть ли корабль в указанной позиции:
        
            Если корабль подбит, он помечается "O", и игрок уведомляется.
        
            Если атака мимо, игроку сообщается об этом.
    
        Условие победы

            После каждой атаки функция hasShips($grid) проверяет, остались ли у противника корабли.
        
            Игра заканчивается, когда один из игроков потопит все корабли противника.
    
        Отображение сетки

            Функция displayGrid($grid, $x, $y) визуально отображает сетки, показывая попадания и промахи.
