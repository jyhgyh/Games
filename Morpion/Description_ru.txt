Русский:

    Описание проекта

        Этот проект представляет собой реализацию классической игры "Крестики-нолики"
        с использованием HTML, CSS и JavaScript. Игра позволяет двум игрокам соревноваться на сетке 3x3,
        отслеживая счета и указывая текущего игрока.

    Обзор кода

        Структура HTML: HTML-файл определяет интерфейс игры, включая сетку, очки игроков и элементы управления.

            Заголовок: Содержит название игры <h1>Morpion</h1> и <div> для отображения текущего игрока (<div id="currentPlayer">Joueur ??</div>).

            Игровая сетка: Сетка 3x3, созданная с помощью вложенных элементов <div> с классами row и cell. Каждая cell представляет клетку на игровом поле.

            Отображение победы: Элемент <span> с классом win-display для отображения победившего игрока (<span class="win-display">Joueur ?? a gagné !</span>).

            Элементы управления: Включают отображение очков игроков и кнопку "Rejouer".

                Счета: Два элемента <div>, отображающие "Joueur 1" и "Joueur 2" с их соответствующими счетами (<span id="playerOne">0</span> и <span id="playerTwo">0</span>).

                Кнопка перезапуска: Элемент <button> с ID replay для сброса игры.

        Внешние ресурсы:

            Таблица стилей CSS: Подключена через <link rel="stylesheet" href="styles/main.css" /> для стилизации интерфейса игры.

            JavaScript-файл: Подключен через <script type="module" src="scripts/morpion.js" defer></script> для добавления логики игры и интерактивности.

            Favicon: Подключен через <link rel="shortcut icon" href="assets/favicon.ico" />.

    Функциональность

        Динамический индикатор игрока: Отображает, чей сейчас ход.

        Интерактивная сетка: Игроки могут нажимать на клетки, чтобы разместить свои метки (обычно "X" или "O").

        Определение победы: Игра определяет, когда один из игроков выиграл, и отображает сообщение о победе.

        Отслеживание счета: Ведет учет количества побед каждого игрока.

        Функция перезапуска: Позволяет игрокам сбросить игровое поле и начать заново.
