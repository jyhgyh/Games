Русский:

    Описание проекта

        Запустите quizz открыв файл "index.html"

        Этот проект представляет собой интерактивное приложение-викторину, разработанное с использованием 
        HTML, CSS и JavaScript.
        Викторина предлагает пользователю серию вопросов, собирает их ответы,
        вычисляет счет и отображает итоговый результат в конце.

    Обзор кода

        JavaScript код (script.js):

            Массив вопросов: Константа questions содержит несколько объектов вопросов. Каждый объект включает:

                "question": Вопрос викторины в виде строки.

                "reponses": Массив возможных ответов.
                
                "solution": Индекс правильного ответа в виде строки (начиная с "1").

        DOM элементы:

            headerContainer: Ссылка на элемент с ID header, где отображается заголовок вопроса.

            listContainer: Ссылка на элемент с ID list, где отображаются варианты ответов.
            
            submitBtn: Ссылка на кнопку отправки с ID submit.

        Переменные:

            score: Отслеживает счет пользователя.

            questionIndex: Отслеживает индекс текущего вопроса.
            
        Слушатели событий:

            submitBtn.onclick = checkAnswer;: Добавляет обработчик события клика на кнопку отправки,
            вызывающий функцию checkAnswer.

        Функции:

            clearPage(): Очищает содержимое headerContainer и listContainer.

            showQuestion(): Отображает текущий вопрос и варианты ответов. Она:

                Использует шаблон для вставки заголовка вопроса в HTML.

                Перебирает массив reponses для создания элементов списка для каждого варианта ответа.
                
                Обновляет listContainer вариантами ответов.

            checkAnswer(): Проверяет выбранный пользователем ответ. Она:

                Получает выбранный радио-кнопкой ответ.
                
                Сравнивает ответ пользователя с правильным решением.
                
                Увеличивает score, если ответ верный.
                
                Определяет, есть ли еще вопросы для отображения или показывает итоговый результат.

            showResult(): Отображает итоговый счет пользователю. Она:

                Создает шаблон результата с учетом счета пользователя.

                Обновляет headerContainer с финальным сообщением.

    Структура HTML (index.html):

        Основные элементы:

            <main>: Оборачивает все приложение викторины.

                <div class="quiz" id="quiz">: Основной контейнер для викторины.

                Заголовок (<div class="quiz_header" id="header">): Отображает заголовок вопроса и плейсхолдеры для сообщений результата.

                Список (<ul class="quiz_list" id="list">): Содержит список вариантов ответов.

                Кнопка отправки: <button class="quiz-submit submit" id="submit">Ответить</button>.

            Скрипты:

                <script src="js/script.js"></script>: Подключает JavaScript файл.

    Функциональность

        Динамическое отображение вопросов: Вопросы и варианты ответов загружаются динамически из массива questions.

        Взаимодействие с пользователем: Пользователи выбирают ответ и отправляют его, чтобы перейти к следующему вопросу.

        Вычисление счета: Приложение отслеживает счет пользователя на протяжении всей викторины.

        Отображение итогового результата: В конце викторины отображается общий счет пользователя.
